This technical guide explains how to access vehicle data through the CAN bus and OBD-II port, decode standardized diagnostic parameters, and stream that data to the cloud using the Blues Notecard. It clarifies the difference between OBD-II and CAN bus and provides code examples for reading frames and monitoring engine metrics like RPM and coolant temperature.
